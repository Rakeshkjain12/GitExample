Git Example

All Git Commands:

  • git init
  • git config --global user.name [username]
  • git conf ig --global user.email [email]
  • git status
  • git add [filename]
  • git add .
  • git commit -m [message]
  • git remote add origin [url]
  • git git push origin master
  • git git remote -v
  • git git log
  • git clone [url]
  • git pull origin master
